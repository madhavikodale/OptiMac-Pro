# OptiMac Pro

🖥️ **Premium macOS System Monitor & Optimizer**

A beautiful, futuristic system monitoring app built with Tauri + React + Tailwind CSS.

## Features

✨ **Real-time System Monitoring**
- CPU, Memory, Disk usage tracking
- Network activity monitoring
- Process management
- Battery & Power status
- System temperature

🎨 **Beautiful UI**
- Dark/Light mode toggle
- Glassmorphism design
- Smooth animations
- macOS-native feel
- Responsive layout

⚙️ **Premium Experience**
- Zero latency updates
- Cyberpunk aesthetic
- Holographic HUD style
- Live metrics dashboard

## Downloads

📥 **[Download OptiMac Pro v1.0.0 (macOS)](https://github.com/madhavikodale/OptiMac-Pro/releases)**

## Tech Stack

- **Frontend:** React 18, Framer Motion, Tailwind CSS
- **Backend:** Rust, Tauri 2
- **Build:** Vite, Tauri Build

## Installation

1. Download the `.dmg` file
2. Open and drag OptiMac Pro to Applications
3. Launch from Applications folder
